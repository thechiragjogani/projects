Lists in HTML

There are two types of lists in HTML.

1. Ordered List

We can create an ordered list in html using ol tag. And list elements in ordered list using li tag.

Example of an ordered list is given below:
  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

2. Un-Ordered List

We can create an un-ordered list in html using ul tag. And list elements in un-ordered list using li tag.

Example of an un-ordered list is given below:
  • Item 1
  • Item 2
  • Item 3
  • Item 4
